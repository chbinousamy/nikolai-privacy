Politique de confidentialité

1. Collecte des données

Cette application ne collecte aucune donnée personnelle. Toutes les informations (comme les scores) sont stockées localement et temporairement pendant l'utilisation de l'app sur votre appareil. Elles ne sont pas transmises à des serveurs externes.

2. Utilisation des données

Les données locales sont utilisées uniquement pour améliorer votre expérience de jeu durant la partie.

3. Partage des données

Aucune donnée n’est partagée avec des tiers.

4. Sécurité

Les données locales sont protégées par les mécanismes de sécurité standard d’iOS.

5. Vos droits

Vous pouvez supprimer les données locales en désinstallant l’application.
